Оптические системы предотвращения перелива

Самым эффективный инструмент для обеспечения безопасности при нижнем наливе. Оптические сенсоры срабатывают в случае превышения заданного уровня нефтепродуктов в отсеке и автоматически отключают насос. По стандартам API, Системы предотвращения перелива обязаны иметь функцию самодиагностики.

Таким образом, оператор будет уверен, что система функционирует правильно еще перед началом загрузки.

Оборудование для нижнего налива

Оборудование терминала и автоцистерны под использование системы нижнего налива устанавливается под стандарты API. Стандарты разработаны американской некоммерческой организацией American Petroleum Institute, созданные для того, чтобы упорядочить процессы в добыче и переработке нефти. Благодаря единому стандарту, все производители выпускают системы совместимого типа и у потребителя появляется широкий выбор нижнего налива.

Терминал нижнего налива оборудуется из двух составных частей — сама установка и API муфта, при помощи которых будет осуществляться закачка в цистерну.

1

Существует два типа: установки с неподвижными металлическими трубами и комбинированные установки в которые входят API муфты, гибкие шланги, шарнирных механизмов, которые в свою очередь упрощают процесс состыковки, а также позволяют перекрещивать шланги и проводить,налив через несколько отсеков одновременно.

2

API муфта — вторая составная часть терминала расположена на конце наливной установки. Ее форма сопоставима со всеми адаптерами на цистернах, оборудованных нижним наливом по стандартам API. Распространены два типа муфт — ручные, где оператор фиксирует рычажный механизм и полуавтоматические, которые обеспечивают сухое соединение.
