Преимущества систем нижнего налива:

Клапан воздушного переключения

Клапан воздушного переключения используются в комплекте с API адаптерами, он выполняет важную функцию: во время соединения он активирует стояночный тормоз автоприцепа или блокирует включение двигателя.

Клапаны воздушного переключения используются вместе с адаптерами рециркуляции ПВС. Во время установки адаптер рециркуляции ПВС, муфта отвода ПВС вдавливает поршень клапана воздушного переключения аналогично гравитационному переходнику.

API адаптер

API адаптер — главный компонент системы нижнего налива, его размеры строго регламентированы для точной совместимости. Конструкция адаптера существует в двух вариантах — открытого типа и гравитационного.

На открытом адаптере состыковка происходит вручную. На гравитационном для соединения необходим переходник, в который входит выжимное устройство.

Пневматические донные клапаны

Пневматические донные клапаны — исключают угрозу разлива, они имеют специальный проток, по которой происходит скол нижней части клапана в случае аварии. Таким образом произойдет потеря только части горючего в трубе.

Блок управления донными клапанами предназначен для открытия клапанов рециркуляции ПВС и пневматических донных клапанов.

Пылезащитные крышки

Пылезащитные крышки — специально сконструированы для защиты адаптеров от вредных веществ и повреждений в ходе эксплуатации, устройство крышки предотвращает случайное открытие.
